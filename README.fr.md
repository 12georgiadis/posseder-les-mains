# Posséder les mains, louer le cerveau

Infrastructures souveraines pour la création avec l'intelligence artificielle. Ce dépôt formalise une méthode que je mets en place sur le terrain, dans le cadre de deux résidences de création soutenues par le CNC ([appel IAG](https://www.cnc.fr/professionnels/actualites/quatre-structures-laureates-de-lappel-a-projets--accompagner-les-createurs-dans-des-usages-exploratoires-des-intelligences-artificielles-generatives-iag_2577815)) : comment donner à des artistes des moyens de production numériques qu'ils possèdent, du libre par défaut, des coûts et des empreintes visibles par geste, et un quotidien trivial (le matin, on ouvre, on retrouve tout).

- **[Le Manifeste](MANIFESTE.md)** : la position. Le degré de création dépend de ce que l'on possède ; l'écologie sans fable (bas-carbone ne veut pas dire propre) ; mesurer, calculer, estimer, questionner.
- **[Le Guide](GUIDE.md)** : le blueprint générique. Architecture en quatre briques, réseau, pile logicielle libre, crédits nominatifs, méthode carbone, prérequis, budget par blocs, leçons.
